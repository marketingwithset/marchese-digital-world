# SET Universe — Chris Marchese Digital World

> An immersive, walkable 3D website experience. Capital, Art, Fashion — all inside a cinematic Three.js environment.

**Built by WAVMVMT × SET Enterprises**

## Stack
- **3D Engine:** Three.js r183+
- **Build Tool:** Vite 6
- **Animation:** GSAP + Custom GLSL Shaders
- **Backend:** Supabase
- **Hosting:** Vercel

## Controls
- **WASD** — Move Chris
- **Mouse** — Orbit camera
- **Scroll** — Zoom in/out
- **E / Click** — Interact with objects
- **Shift** — Sprint

## Rooms
- Main Hall (Grand Lobby) — Light Pillars, Art, Sculptures, Phone Booth
- Capital Portal — Executive boardroom, deal flow
- Infrastructure Portal — Systems, team bios
- Growth Portal — Campaigns, CMO showcase
- Art Gallery — Chris Marchese originals
- Fashion Zone — Runway, try-on, checkout
- Media Lounge — Red carpet, press wall
- Car Showroom — Chris's car

## Development
```bash
npm install
npm run dev
```
